# Customer & Churn Analysis 📊

Analyzed customer behavior, subscription trends, and churn patterns using Python, Pandas, Matplotlib, and Seaborn.

## 🔍 What I Worked On
- Data loading and preprocessing from multiple datasets (Customers, Churn, Subscriptions, Transactions)
- Data filtering and segmentation (region-wise, active vs inactive customers)
- Exploratory Data Analysis (EDA) using visualizations
- Customer churn analysis and comparison
- Subscription trend analysis (Annual vs Monthly)
- Transaction analysis based on spending patterns

## 📈 Steps Involved

**1. Data Loading**
Imported libraries and loaded 4 datasets: Customers, Churn, Subscriptions, Transactions

**2. Data Exploration**
Viewed datasets, selected key columns, filtered customers by region

**3. Customer Analysis**
Region-wise distribution (pie chart), active vs inactive customers visualization

**4. Subscription Analysis**
Compared Annual vs Monthly plans using bar charts

**5. Transaction Analysis**
Filtered transactions above $100, created `Amount_category` column, built pie chart, bar chart, and scatter plot

**6. Churn Analysis**
Compared churned vs retained customers using bar chart

**7. Data Merging**
Merged customers + subscriptions to find active customers on annual plans
